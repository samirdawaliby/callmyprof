1. Workflow : Client depuis le site (Organique)

Ce workflow simule un client qui arrive sur callmyprof.com et remplit le formulaire.

1 Remplir le formulaire CTA

  1. Ouvrir https://callmyprof.com
  2. Tester la langue : cliquer EN / FR / AR dans la navbar
  3. Scroller vers le formulaire (hero section)
  4. Remplir : Prenom, Telephone, Email, Matiere, Niveau
  5. Cliquer "Book Your FREE Trial Lesson"
  6. Verifier la redirection vers /thanks
✅ Resultat attendu

Email de confirmation envoye au client + notification admin a contact@callmyprof.com. Le lead apparait dans le CRM admin (/leads).

2 Verifier dans le CRM admin

  1. Aller sur callmyprof.com/login
  2. Se connecter : admin@callmyprof.com / admin123
  3. Aller dans Leads (sidebar)
  4. Verifier que le nouveau lead apparait avec le badge Organic
  5. Cliquer sur le lead pour voir le detail

3 Envoyer une invitation booking

  1. Sur la page detail du lead, cliquer "Send Booking Invite"
  2. Un email avec un lien unique est envoye au client
  3. Le statut du lead passe a Contacted
  4. Verifier l'email recu (sujet: "Book Your Free Consultation")

4 Reserver un creneau (cote client)

  1. Ouvrir le lien du booking email (/book/{token})
  2. Choisir une date dans la grille
  3. Choisir une heure
  4. Confirmer la reservation
✅ Resultat attendu

Video room creee automatiquement (Daily.co ou Jitsi). Email de confirmation envoye avec lien visio + lien portail etudiant. Lead passe en "Qualified".

5 Portail etudiant

  1. Ouvrir le lien "Mon Dashboard" dans l'email de confirmation
  2. Le portail affiche : carte de bienvenue, sessions a venir, profil, support
  3. Cliquer "Join" pour rejoindre la session video
  4. Tester le lien Discord dans la section support

6 Rejoindre la session video

  1. Depuis le portail etudiant, cliquer "Join" sur la session
  2. La page /session/{id} s'ouvre avec l'iframe video embedded
  3. Le header affiche le nom, la date, l'heure, le badge Live
  4. L'URL reste callmyprof.com (pas de jitsi ou daily visible)
